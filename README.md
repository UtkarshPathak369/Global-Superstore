# Global Superstore Sales Insights

This project analyzes the Global Superstore dataset using Python and SQL to provide business insights.

## Features:
- Total Sales by Region (Bar Chart)
- Total Profit by Category (Bar Chart)
- Top 5 Products by Sales (Bar Chart)
- SQL queries for key business metrics

## Tools Used:
- Python (pandas, matplotlib)
- MySQL
- GitHub

## Objective:
To demonstrate data analysis and visualization skills for business insights, making it resume-ready for data/analytics roles.
