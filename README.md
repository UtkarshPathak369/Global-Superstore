# Sales Insights using Global Superstore Data

## Project Overview
This project analyzes sales data from the Global Superstore dataset to extract
meaningful business insights using SQL. It focuses on data warehousing concepts
and data analysis for decision-making.

## Tools Used
- MySQL
- SQL
- GitHub

## Dataset
Global Superstore sales dataset in CSV format.

## Key Analysis
- Region-wise sales performance
- Top selling products
- Profit and discount analysis
- Monthly and yearly sales trends
- Customer segmentation

## How to Use
1. Import the CSV file into MySQL
2. Execute SQL queries from the analysis SQL file
3. Analyze the results for business insights

## Outcome
This project demonstrates practical SQL skills, data analysis,
and data warehousing concepts used in real-world scenarios.# Global Superstore
